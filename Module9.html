Module9: IPv4 Addressing

គោលបំណងមេរៀន (Module Objective) គណនាការបែងចែក Subnet របស់ IPv4 ដើម្បីបែងចែកបណ្តាញ (Network) ឱ្យមានប្រសិទ្ធភាព។

១. តើ IPv4 ជាអ្វី?

IPv4 (Internet Protocol Version 4) គឺជាប្រព័ន្ធអាសយដ្ឋាន (Addressing System) ដែលប្រើសម្រាប់កំណត់អត្តសញ្ញាណឧបករណ៍នៅលើបណ្ដាញ (Network) និងអ៊ីនធឺណិត។

ឧទាហរណ៍ IPv4 Address៖

  • 192.168.1.1
  • 172.16.0.1
  • 10.0.0.1
  • 8.8.8.8

IPv4 មានប្រវែង 32-bit ហើយត្រូវបានបែងចែកជា 4 Octets (មួយ Octet = 8 Bits)។ ឧទាហរណ៍៖

  • 192.168.1.1
  • 192 .168 .1 .1
  • 8bit 8bit 8bit 8bit

២. រចនាសម្ព័ន្ធ IPv4 Address

IPv4 Address មាន ២ ផ្នែក៖

1. Network ID

សម្គាល់បណ្ដាញ (Network)។

2. Host ID

សម្គាល់ឧបករណ៍ (Computer, Laptop, Phone, Router) នៅក្នុងបណ្ដាញ។ ឧទាហរណ៍៖

  • 192.168.1.10/24
  • Network ID = 192.168.1
  • Host ID = 10

៣. Private IP Address

Private IP ប្រើសម្រាប់បណ្ដាញខាងក្នុង (LAN)។

សង្ខេប

  • IPv4 មាន 32 Bits
  • IPv4 មាន 4 Octets
  • មាន Network ID និង Host ID
  • Private IP ប្រើក្នុង LAN
  • Public IP ប្រើលើ Internet
  • Subnet Mask ប្រើបែងចែក Network និង Host
  • CIDR Notation ដូចជា /24, /16, /8
  • IPv4 មាន Class A, B, C, D, E

IPv4 Address Classes

Class Range Default Mask
A 1-126 255.0.0.0

IPv4 ជាអ្វី?

IPv4 (Internet Protocol Version 4) គឺជាអាសយដ្ឋានដែលប្រើសម្រាប់កំណត់អត្តសញ្ញាណឧបករណ៍នៅលើបណ្តាញ។

ឧទាហរណ៍៖

  • 192.168.1.10
IPv4 មាន 32 Bits ហើយបែងចែកជា 4 Octets (8 Bits ក្នុងមួយ Octet)។
  • 192 . 168 . 1 . 10
