Shortcut ប្រើញឹកញាប់បំផុត

  1. Ctrl + N → បង្កើត File ថ្មី
  2. Ctrl + S → Save
  3. Ctrl + Z → Undo
  4. Ctrl + T → កែទំហំរូបភាព
  5. Ctrl + J → ចម្លង Layer
  6. B → Brush Tool
  7. V → Move Tool
  8. T → Type Tool
  9. C → Crop Tool
  10. Z → Zoom Tool
  11. Ctrl + D → ដោះ Selection
  12. Ctrl + T Free Transform (ប្តូរទំហំ/បង្វិល)
