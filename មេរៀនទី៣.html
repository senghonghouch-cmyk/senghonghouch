មេរៀនទី៣ មូលដ្ឋានគ្រឹះ C++

តើ C++ ជាអ្វី?

C++ គឺជាភាសាកម្មវិធី (Programming Language) ដែលប្រើសម្រាប់បង្កើត Software, Game, Application និង System Programs។

1. មូលដ្ឋានគ្រឹះ C++

  • C++ ជាភាសាសរសេរកម្មវិធី។
  • Syntax = ច្បាប់សរសេរកូដ។
  • cout = បង្ហាញទិន្នន័យ។
  • cin = បញ្ចូលទិន្នន័យ។
  • Variables = ផ្ទុកទិន្នន័យ។
  • Data Types = int, float, double, char, string, bool
  • Operators = +, -, *, /, ==, >, <
  • Strings = អក្សរ/អត្ថបទ។
  • Math = sqrt(), pow()
  • Boolean = true, false

2. លក្ខខណ្ឌ (Conditions)

  • if = ពិនិត្យលក្ខខណ្ឌមួយ។
  • if...else = មានជម្រើស 2។
  • else if = លក្ខខណ្ឌច្រើន។
  • switch = ជ្រើសរើសតម្លៃជាច្រើន។

3. រង្វិលជុំ (Loops)

  • while = រត់ពេលលក្ខខណ្ឌពិត។
  • do...while = រត់យ៉ាងតិច 1 ដង។
  • for = រត់ចំនួនដងកំណត់។
  • break = បញ្ឈប់ Loop។
  • continue = រំលងជុំបច្ចុប្បន្ន។

4. Arrays

  • 1D Array = បញ្ជីទិន្នន័យមួយជួរ។
  • 2D Array = បញ្ជីទិន្នន័យជាតារាង។
  • Array Operations = បន្ថែម, បង្ហាញ, ស្វែងរកទិន្នន័យ។

5. Functions

  • Function = ក្រុមកូដធ្វើការងារមួយ។
  • Parameters = តម្លៃបញ្ចូល។
  • Return = តម្លៃត្រឡប់។
  • Overloading = Function ឈ្មោះដូចគ្នា តែ Parameter ខុសគ្នា។
  • Recursion = Function ហៅខ្លួនឯង។
  • Scope = តំបន់ប្រើប្រាស់ Variable។
  • Lambda = Function ខ្លីៗ។

6. Data Types កម្រិតខ្ពស់

  • struct = ប្រមូល Data ច្រើនប្រភេទ។
  • enum = កំណត់តម្លៃថេរ។
  • Reference (&) = ឈ្មោះផ្សេងសម្រាប់ Variable។
  • Pointer (*) = ផ្ទុក Address របស់ Variable។
  • Dynamic Memory = ប្រើ new និង delete គ្រប់គ្រង Memory។
គេហទំព័រ: C++

មេរៀនទី៣៖ មូលដ្ឋានគ្រឹះ C++

១. តើ C++ ជាអ្វី?

C++ គឺជាភាសាសរសេរកម្មវិធី (Programming Language) ដែលត្រូវបានបង្កើតឡើង ដើម្បីអភិវឌ្ឍកម្មវិធីកុំព្យូទ័រ។ C++ ត្រូវបានប្រើសម្រាប់បង្កើត Software, Game, Operating System, Mobile Application និងកម្មវិធីផ្សេងៗជាច្រើន។ ភាសានេះមានល្បឿនលឿន និងមានប្រសិទ្ធភាពខ្ពស់។

២. Syntax

Syntax គឺជាច្បាប់ ឬរបៀបសរសេរកូដឱ្យបានត្រឹមត្រូវ។ ប្រសិនបើសរសេរខុសពីច្បាប់ កម្មវិធីនឹងបង្ហាញកំហុស (Error) ហើយមិនអាចដំណើរការបានឡើយ។ 

#include <iostream>
using namespace std;

int main() {
    cout << "Hello World";
    return 0;
}

៣. cout

cout (Character Output) ត្រូវបានប្រើសម្រាប់បង្ហាញទិន្នន័យ ឬសារផ្សេងៗទៅលើអេក្រង់។ សញ្ញា << ត្រូវបានប្រើដើម្បីបញ្ជូន ទិន្នន័យទៅ cout។ 

cout << "Welcome to C++";
cout << 100;

៤. cin

cin (Character Input) ត្រូវបានប្រើសម្រាប់ទទួលទិន្នន័យពីអ្នកប្រើប្រាស់ តាមរយៈ Keyboard។ សញ្ញា >> ត្រូវបានប្រើដើម្បីផ្ទុកតម្លៃ ដែលអ្នកប្រើបានបញ្ចូល។ 

int age;
cin >> age;

៥. Variables

Variable គឺជាទីតាំងមួយនៅក្នុង Memory ដែលប្រើសម្រាប់ផ្ទុកទិន្នន័យ។ Variable នីមួយៗត្រូវមានឈ្មោះ និងប្រភេទទិន្នន័យច្បាស់លាស់។ 

int age = 20;
string name = "Dara";

៦. Data Types

Data Type គឺជាប្រភេទទិន្នន័យដែល Variable អាចផ្ទុកបាន។ ការជ្រើសរើស Data Type ត្រឹមត្រូវនឹងជួយឱ្យកម្មវិធីប្រើ Memory បានមានប្រសិទ្ធភាព។

Data Type ការប្រើប្រាស់ ឧទាហរណ៍
int ចំនួនគត់ 10, 50
float លេខទសភាគ 3.14
double លេខទសភាគមានភាពត្រឹមត្រូវខ្ពស់ 3.141592
char អក្សរមួយតួ 'A'
string អត្ថបទ "Cambodia"
bool តម្លៃពិត ឬមិនពិត true, false

៧. Operators

Operators គឺជាសញ្ញាដែលប្រើសម្រាប់ធ្វើប្រតិបត្តិការលើទិន្នន័យ។

Arithmetic Operators

  • + បូក
  • - ដក
  • * គុណ
  • / ចែក
  • % រកសំណល់
int a = 10, b = 3;
cout << a + b;
cout << a % b;

Comparison Operators

  • == ស្មើ
  • != មិនស្មើ
  • > ធំជាង
  • < តូចជាង
  • >= ធំជាង ឬស្មើ
  • <= តូចជាង ឬស្មើ

៨. Strings

String គឺជាប្រភេទទិន្នន័យដែលប្រើសម្រាប់ផ្ទុកអក្សរ ឬអត្ថបទច្រើនតួ។ 

string school = "ABC School";
cout << school;

៩. Math Functions

C++ មានអនុគមន៍គណិតវិទ្យាជាច្រើនដែលស្ថិតនៅក្នុង Library <cmath> ។

  • sqrt() = រកឫសការ៉េ
  • pow() = គណនាស្វ័យគុណ
#include <cmath>

cout << sqrt(25);     // 5
cout << pow(2, 3);    // 8

១០. Boolean

Boolean គឺជាប្រភេទទិន្នន័យដែលមានតម្លៃតែ ២ ប៉ុណ្ណោះ គឺ true (ពិត) និង false (មិនពិត)។ វាត្រូវបានប្រើជាញឹកញាប់នៅក្នុងការសម្រេចចិត្ត និងលក្ខខណ្ឌផ្សេងៗ។ 

bool isStudent = true;
bool hasLicense = false;

cout << isStudent;
cout << hasLicense;

មូលដ្ឋានគ្រឹះ C++ រួមមាន Syntax, cout, cin, Variables, Data Types, Operators, Strings, Math Functions និង Boolean។ ការយល់ដឹងពីមេរៀនទាំងនេះ គឺជាជំហានដំបូងដ៏សំខាន់ សម្រាប់ការសិក្សាភាសា C++ និងការអភិវឌ្ឍកម្មវិធីកុំព្យូទ័រ នៅកម្រិតខ្ពស់បន្តទៅមុខទៀត។
