មេរៀនទី២ Workspace

១. តើ Workspace ជាអ្វី?

Workspace គឺជាកន្លែងធ្វើការនៅក្នុងកម្មវិធី Adobe Photoshop ដែលមានការរៀបចំឧបករណ៍ (Tools), ផ្ទាំងបញ្ជា (Panels) និងម៉ឺនុយផ្សេងៗ ដើម្បីជួយឱ្យអ្នកប្រើប្រាស់អាចធ្វើការកែសម្រួលរូបភាពបានយ៉ាងងាយស្រួល និងមានប្រសិទ្ធភាព។

២. ផ្នែកសំខាន់ៗនៃ Workspace

១. Menu Bar

ស្ថិតនៅផ្នែកខាងលើបំផុតនៃកម្មវិធី។ វាមានពាក្យបញ្ជាដូចជា៖

  • File → បង្កើត ឬបើកឯកសារ
  • Edit → កែប្រែ និងកំណត់ផ្សេងៗ
  • Image → កែតម្រូវរូបភាព
  • Layer → គ្រប់គ្រង Layer
  • Type → បង្កើតអត្ថបទ
  • Select → ជ្រើសរើសតំបន់
  • Filter → ដាក់ Effect
  • View → គ្រប់គ្រងការមើល
  • Window → បើក/បិទ Panels
  • Help → ជំនួយ

២. Options Bar

ស្ថិតនៅខាងក្រោម Menu Bar។

  • បង្ហាញជម្រើសរបស់ Tool ដែលកំពុងប្រើ។
  • ជម្រើសនឹងផ្លាស់ប្តូរតាម Tool នីមួយៗ។

៣. Tools Panel

ស្ថិតនៅខាងឆ្វេងនៃ Workspace។ ឧបករណ៍សំខាន់ៗមាន៖

  • Move Tool
  • Marquee Tool
  • Lasso Tool
  • Crop Tool
  • Brush Tool
  • Eraser Tool
  • Gradient Tool
  • Text Tool
  • Shape Tool
  • Zoom Tool

៤. Document Window

ជាកន្លែងដែលបង្ហាញរូបភាព ឬឯកសារដែលកំពុងធ្វើការ។ អ្នកអាច៖

  • កែសម្រួលរូបភាព
  • គូររូប
  • បន្ថែមអត្ថបទ
  • មើលលទ្ធផលនៃការងារ

៥. Panels

ស្ថិតនៅផ្នែកខាងស្តាំ។ Panels ដែលប្រើញឹកញាប់មាន៖

  • Layers Panel → គ្រប់គ្រង Layer
  • Properties Panel → បង្ហាញលក្ខណៈសម្បត្តិ
  • Color Panel → ជ្រើសរើសពណ៌
  • History Panel → មើលជំហានការងារត្រឡប់ក្រោយ
  • Adjustments Panel → កែតម្រូវពន្លឺ និងពណ៌

៣. ប្រភេទ Workspace នៅក្នុង Photoshop

Photoshop មាន Workspace ដែលបានកំណត់រួចជាស្រេច ដូចជា៖

  1. Essentials – Workspace ស្តង់ដារ
  2. Photography – សម្រាប់កែសម្រួលរូបថត
  3. Graphic and Web – សម្រាប់ការរចនាគេហទំព័រ
  4. Painting – សម្រាប់គំនូរឌីជីថល
  5. Motion – សម្រាប់ Animation និង Video

៤. របៀបផ្លាស់ប្តូរ Workspace

  1. ចូលទៅកាន់ Window
  2. ជ្រើស Workspace
  3. ជ្រើស Workspace ដែលចង់ប្រើ
  4. ប្រសិនបើចង់ត្រឡប់ទៅដើម សូមជ្រើស Reset Essentials

៥. សារៈសំខាន់នៃ Workspace

  • ជួយរៀបចំឧបករណ៍ឱ្យមានរបៀបរៀបរយ។
  • បង្កើនល្បឿនការងារ។
  • ងាយស្រួលស្វែងរកឧបករណ៍។
  • អាចកែប្រែតាមតម្រូវការរបស់អ្នកប្រើប្រាស់។

សង្ខេប

Workspace គឺជាបរិយាកាសធ្វើការរបស់ Adobe Photoshop ដែលរួមមាន Menu Bar, Options Bar, Tools Panel, Document Window និង Panels ផ្សេងៗ។ ការយល់ដឹងអំពី Workspace នឹងជួយឱ្យអ្នកប្រើ Photoshop បានយ៉ាងមានប្រសិទ្ធភាព និងងាយស្រួលជាងមុន។
