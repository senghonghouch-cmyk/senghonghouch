១. តើ HTML ជាអ្វី?

HTML មានន័យថា HyperText Markup Language។ វាជាភាសាដែលប្រើសម្រាប់បង្កើត និងរៀបចំមាតិកានៅលើគេហទំព័រ។HTML (HyperText Markup Language) គឺជាភាសាសម្គាល់ (Markup Language) ដែលប្រើសម្រាប់បង្កើត និងរៀបចំគេហទំព័រ (Web Pages)។ HTML ជាឆ្អឹងខ្នងរបស់គេហទំព័រទាំងអស់ ដែលប្រាប់ Browser អំពីរបៀបបង្ហាញអត្ថបទ រូបភាព តារាង ប៊ូតុង និងធាតុផ្សេងៗ។

ឧទាហរណ៍៖ នៅពេលអ្នកចូលមើលគេហទំព័រ Browser ដូចជា Chrome, Firefox ឬ Edge វានឹងអាន HTML Code ហើយបង្ហាញជាទំព័រដែលអ្នកឃើញ។
