ក្នុងប្រអប់ Format: ជ្រើសរើសយកប្រភេទ File ណាមួយដែលត្រូវការ

  • BMP (BMP, RLE, "DIB): ជាប្រភេទរូបភាពដែលមានទំហំតូចៗ
  • CompuServe GIF (GIF) : ជាប្រភេទរូបភាពដែលមានទំហំតូចៗ ដូច BMP ដែរ
  • JPEG (.JPG, JPEG, JPE): ជាប្រភេទរូបភាពសំរាប់ប្រើប្រាស់ទូទៅ
  • Photoshop PDF (*.PDF, *.PDP): ជាប្រភេទរូបភាពដែលមាន File th PDF
  • PNG (*.PNG): ជាប្រភេទរូបភាពដែលគ្មាន Background
  • TIFF (TIF, TIFF): ជាប្រភេទ File Photoshop សំរាប់ប្រើប្រាស់ក្នុងការងារបោះពុម្ព
